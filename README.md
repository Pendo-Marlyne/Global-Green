# Global Green Energy Report

A single-page technical report built with semantic HTML5 and an emerald-themed stylesheet. I

## Files

| File | Purpose |
|------|---------|
| `index.html` | Report markup: `header`, `nav`, `main`, `article`, `section`, `footer`, TOC (`ol`), key takeaways (`ul`), and comparison table |
| `styles.css` | Visual styling: emerald palette, typography, table and navigation presentation |

## How to view

Open `index.html` in a web browser
